# Quiz Game — Обзор проекта

## Суть
Приложение для проведения квизов с вариантами ответа. Ведущий управляет квизом через веб-интерфейс (экран + личный кабинет), участники играют через Telegram-бота.

## Стек технологий

| Компонент       | Технологии                          |
|-----------------|-------------------------------------|
| Backend API     | Go (Gin + GORM)                     |
| Frontend        | React + Redux + React Router        |
| Telegram Bot    | Python (aiogram)                    |
| База данных     | PostgreSQL                          |
| Контейнеризация | Docker Compose                      |

## Архитектура

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  React SPA   │────▶│   Go API     │◀────│  Aiogram Bot  │
│  (Frontend)  │     │  (Backend)   │     │  (Python)     │
└──────────────┘     └──────┬───────┘     └──────────────┘
                            │
                     ┌──────▼───────┐
                     │  PostgreSQL   │
                     └──────────────┘
```

- **React SPA** — веб-интерфейс для ведущего (личный кабинет + экран сессии).
- **Go API** — единый бэкенд, обрабатывает всю бизнес-логику, хранение данных.
- **Aiogram Bot** — Telegram-бот для участников; работает как отдельный фронтенд, общается только с Go API (не имеет прямого доступа к БД).
- **PostgreSQL** — единая база данных, доступ только через Go API.

## Ключевые принципы
1. Бот **не** работает с БД напрямую — только через API.
2. Авторизация ведущего — через веб-интерфейс (логин/пароль).
3. Авторизация участника — автоматическая через Telegram ID + ник, введённый при входе в сессию.
4. Очки: `100` за правильный ответ + бонус от `1` до `50` в зависимости от порядка ответа (чем быстрее — тем больше бонус).
